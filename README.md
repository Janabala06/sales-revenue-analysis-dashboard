# Sales & Revenue Analysis Dashboard

This project is a Power BI dashboard created as part of a Data Analytics internship task.

## Features
- Total Revenue KPI
- Revenue Trend Over Time
- Top Performing Products
- Interactive Region Filter

## Tools Used
- Power BI Desktop
- CSV data

## Objective
To analyze sales performance, track key metrics, and generate basic business insights using data visualization.
